Setting Up Alerts
Step-by-step guide for TradingView alerts
① Load the Indicator
Go to "Indicators" → "Invite-only" → select the BRH Indicator and add it to your chart.
② Create an Alert
Right-click on the chart → "Add Alert" or click the clock icon in the top right.
③ Set the Condition
Select the BRH Indicator as the source → set the condition to "alert() function calls only".
④ Enable Notifications
Check app notification, email or SMS → save the alert. Done!
